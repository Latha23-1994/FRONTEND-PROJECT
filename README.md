# 🏢 JPR Facility Management Service Website

A modern, responsive frontend website for a Facility Management company built using **HTML, CSS, Bootstrap 5, and JavaScript**.

This project includes a complete landing page, service showcase with interactive hover effects, and a Client Portal with Login & Register functionality using LocalStorage authentication.

---

## 🚀 Live Features

- ✅ Responsive Navigation Bar
- ✅ Hero Section with Branding
- ✅ Animated Service Cards with Large Hover Effects
- ✅ Client Login System
- ✅ Client Registration System
- ✅ LocalStorage-based Authentication
- ✅ Smooth Scrolling Navigation
- ✅ Fully Responsive Design (Mobile Friendly)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5.3
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Google Fonts
